标准C10-预处理

发表于 2017-09-24 | 分类于 C语言基础 | 阅读次数
| 字数统计 4,347 | 阅读时长 16

ANSI C 标准规定可在 C 源程序中加入“预处理命令”，用来改进程序设计环境，提高编程效率。
预处理命令不是 C 语句，不能直接对它进行编译，必须在对程序编译之前，对这些特殊命令做“预处理”。
若用#define 命令定义一个符号常量 A，则预处理时会将程序中所有的 A 都置换为指定的字符串；若程序中用#include 命令包含一个头文件，则预处理时会用头文件中的实际内容代替该命令。经过预处理后的程序就不再包括预处理命令了，最后再由编译程序对预处理后的源程序进行编译处理，进而得到可执行目标代码。
C 提供的预处理功能主要有 3 种：宏定义；文件包含；条件编译。这些命令都以”#”开头。

标准C06-深入理解函数

发表于 2017-08-27 | 分类于 C语言基础 | 阅读次数
| 字数统计 7,521 | 阅读时长 28

函数是C语言中模块化程序设计的最小单位，是构成程序的基本模块；
程序的执行从main()的入口开始，到main()的出口结束；
从使用者的角度将函数分为标准库函数和自定义函数两种；
使用ANSI C的库函数必须在程序的开头把函数所在的头文件包含进来；
函数在使用之前必须先声明；
函数名表示符的命名规则与变量的命名规则相同；
函数体内部定义的变量(内部变量)只能在该函数内部访问，函数头部参数表里的变量(形式参数)也是内部变量；
主调函数把实参的值赋值给被调函数的形参的过程称为参数传递；
函数可以有多个return语句，但返回值只能有一个，类型是除数组以外任何类型；
当函数返回值类型定义为void类型时，表示函数没有返回值，函数可以没有return语句，也可以有”return;”；
print函数是有返回值的，返回值就是print函数输出的字符个数；
