标准C09-结构体

| 字数统计 964 | 阅读时长 4

在编程语言中，最基本的，不可再分的数据类型称为基本类型，例如整型，浮点型；根据语法规则由基本类型组合而成的类型称为复合类型，数组就是一种复合类型，这一篇内容将介绍另一种复合类型－结构体。先来讲下数组和结构体概念上的区别：

  • 数组：由多个同类型的数据组合而成，成员数据的类型不一定是基本类型，也可以是自定义类型
  • 结构体：可以由多个不同类型的数据组合而成，成员数据的类型不一定是基本类型，也可以是自定义类型

在编译时，对类型是不分配空间的，而只对变量分配空间。在初始化结构体变量时，可以采用初始化列表对结构体成员依次赋值，且只能对最低级的成员进行赋值或存取以及运算 ，其中未被指定初值的数值型变量默认为0，char 型默认为’\0’，指针类型默认为NULL。

用 typedef 定义类型

typedef 是用来给已存在的类型起一个别名，并不能创造新类型，如：

归纳，声明一个新类型名的方法：
(1) 先按定义变量形式书写，如 int (f)();
(2) 将变量名换成指定的新类型名，如 int (POINTER)();
(3) 在前面加上 typedef，得到 typedef int (*POINTER)();
(4) 用来定义变量，如 POINTER p1;

对比typedef 和#define 的区别，例如：typedef int COUNT;和#define COUNT int;
都是用 COUNT 来代替 int，但二者是不同的，#define 是在预编译时处理的，只能作简单的字符串替；而 typedef 是在编译时处理的，而且不是作简单的字符串替换，而是采用如同定义变量的方法来声明类型。当不同源文件都要用到同一类型数据时，可以用 typedef 声明一些数据类型，并单独的放到一个文件中，在需要用的文件中用#include 包含此文件即可，使用 typedef 有利于程序的通用和移植，在移植时只需改动 typedef 定义体即可，如：typedef int INTEGER; → typedef long INTEGER;
